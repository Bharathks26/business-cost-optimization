# Static Images Folder

Place your custom background image here.

## For Home Page Background:
1. Generate your AI image using DALL-E, Midjourney, or Leonardo.ai with the prompt provided
2. Save the image as `bg.jpg` in this folder
3. The home page will automatically use it

## Current Setup:
- Home page uses: `/static/images/bg.jpg`
- If file doesn't exist, it falls back to the Unsplash URL

## Recommended Image Specs:
- Format: JPG or PNG
- Resolution: 1920x1080 or higher
- Aspect Ratio: 16:9
- File size: Under 500KB (optimized for web)
